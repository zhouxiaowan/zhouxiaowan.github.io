# 10.2 css垂直无限滚动

垂直属入上下滚动，和水平同理，先复制出一组数据，然后向上移一组的高度，只是把对宽度的处理变成了高度

<!-- css垂直滑动 -->
  <div class="vertical">
    <div class="vertical-cont">
      <p>今天是个好日子1</p>
      <p>今天是个好日子2</p>
      <p>今天是个好日子3</p>
      <p>今天是个好日子4</p>
      <p>今天是个好日子5</p>
      <p>今天是个好日子1</p>
      <p>今天是个好日子2</p>
      <p>今天是个好日子3</p>
      <p>今天是个好日子4</p>
      <p>今天是个好日子5</p>
    </div>
  </div>

.vertical{
  width: 100%;
  height: 100px;
  overflow: hidden;
}
.vertical-cont{
  height: 208px; /* 两组数据的高度度 */
  animation: up 6s linear infinite forwards;
}
.vertical-cont:hover{
  animation-play-state:paused;
  -webkit-animation-play-state:paused;
}
@keyframes up {
  from {
    transform: translateY(0);
  }
  to {
    transform: translateY(-104px);  /* 一组数据的高度度 */
  }
}
